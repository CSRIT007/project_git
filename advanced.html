Advanced Ubuntu Commands

1. Show Running Processes

Shows running processes.

ps aux

2. Kill Process

Stops a process by process ID.

kill PID

3. Check Disk Usage

Shows disk usage.

df -h

4. Check Memory Usage

Shows memory usage.

free -h

5. Check Open Ports

Shows listening ports.

sudo ss -tulnp

6. SSH to Server

Connects to another server using SSH.

ssh username@server_ip

7. Check System Logs

Shows system logs.

journalctl -xe

8. Check Firewall Status

Checks Ubuntu firewall status.

sudo ufw status
