Beginner Ubuntu Commands

1. Check Current Folder

Shows your current location in the terminal.

pwd

2. List Files and Folders

Shows all files and folders in the current directory.

ls

3. Change Folder

Moves to another folder.

cd folder_name

4. Create Folder

Creates a new folder.

mkdir myfolder

5. Create File

Creates an empty file.

touch index.html

6. Edit File with Nano

Opens a file using nano editor.

nano index.html

7. Remove File

Deletes a file.

rm index.html

8. Remove Empty Folder

Deletes an empty folder.

rmdir myfolder
