Beginner Ubuntu Commands

1. Check Current Folder

Shows your current location in the terminal.

pwd

2. List Files and Folders

Shows files and folders in the current directory.

ls

3. List All Files

Shows all files, including hidden files.

ls -la

4. Change Folder

Moves to another folder.

cd folder_name

5. Go Back One Folder

Moves back to the parent folder.

cd ..

6. Create Folder

Creates a new folder.

mkdir myfolder

7. Create File

Creates an empty file.

touch index.html

8. Edit File with Nano

Opens a file using nano editor.

nano index.html

9. Remove File

Deletes a file.

rm index.html

10. Remove Empty Folder

Deletes an empty folder.

rmdir myfolder
