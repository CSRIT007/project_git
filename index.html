Ubuntu Basic Commands

This page explains basic Ubuntu commands for beginners. Ubuntu commands are used in the terminal to manage files, folders, software, users, network, and system services.

1. Check Current Folder

This command shows your current location in the terminal.

pwd

2. Show Files and Folders

This command lists files and folders in the current directory.

ls

3. Change Directory

This command moves you into another folder.

cd folder_name

4. Create New Folder

This command creates a new folder.

mkdir myfolder

5. Create New File

This command creates an empty file.

touch index.html

6. Edit File with Nano

This command opens a file using the nano editor.

nano index.html

7. Remove File

This command deletes a file.

rm index.html

8. Remove Folder

This command deletes an empty folder.

rmdir myfolder

9. Update Ubuntu Package List

This command updates the software package list.

sudo apt update

10. Install Software

This command installs software in Ubuntu.

sudo apt install package_name

11. Check IP Address

This command shows your Ubuntu IP address.

hostname -I

12. Check SSH Service

This command checks SSH server status.

sudo service ssh status

13. Start SSH Service

This command starts SSH service.

sudo service ssh start

14. Change Hostname

This command changes your Ubuntu hostname.

sudo hostnamectl set-hostname ubuntu
Note: In Ubuntu, you should use ssh service name, not sshd. Example: sudo service ssh status

15. Check SSH Service Again

This command helps you confirm SSH is running.

sudo service ssh status
Note: If you use Ubuntu WSL, you can start SSH with sudo service ssh start.
