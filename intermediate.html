Intermediate Ubuntu Commands

1. Update Package List

Updates Ubuntu software package list.

sudo apt update

2. Upgrade Packages

Upgrades installed packages.

sudo apt upgrade -y

3. Install Software

Installs software package.

sudo apt install package_name

4. Check IP Address

Shows Ubuntu IP address.

hostname -I

5. Check Network Interfaces

Shows network interface information.

ip addr

6. Change File Permission

Makes a script executable.

chmod +x script.sh

7. Check SSH Service

Checks SSH server status.

sudo service ssh status

8. Start SSH Service

Starts SSH service.

sudo service ssh start
